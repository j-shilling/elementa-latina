Name Running On Description
GWT client GWT compiles java to javascript. All of Element is written in Java; however, not all .java files end up running on the server's JVM. Instead GWT figures out how to generate a front-end base one the java code we provide
GWTP hello client GWTP is an MVP framework for GWT. We use it to enforce better coding practices and to facilitate things like RESTful server calls;
GWT-Jackson client GWTP is pulls in gwt-jackson as a dependency used for JSON serialization. We can also use the same gwt-jackson annotations to customize the serialization of our own objects. This is how we are able to make our objects immutable.
Google GIN client GWTP makes heavy use of Dependency Injection using GIN. GIN is an implementation of Google's Guice, which makes heavy use of java reflection . This is not supported on GWT and so to use Guice-like dependency injection we need to use GIN.
Google Guava client / server Google's Guava is a collection of generally usefull classes. Primarily we are interested in their immutable collections and multimaps.
Google Guice server The sever-side framework for Dependency Injection.
RESTEasy server Our RESTful framework for the backend stuff.
