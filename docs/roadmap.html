Elementa Latina

Elementa Latina is meant to serve as a resource for Latin classrooms and to replace a few existing websites: BlackBoard, Whitaker's Words, and The Perseus Projects. We hope to accomplish the same tasks which these sites propose to, but to do so with more elegance and with a modern design.

Elementa Latina 0.1

The first release of Elementa Latina will be a simple online dictionary. Unlike virtually all current online dictionaries, we will produce a monolingual dictionary, with optional English translations, geared to show people how to use words actively and not simply to passively translate--a version of the Totius Lexicon Latinitatis truly for the 21st century.

The initial release, therefore, needs first to maintain a database of users and authenticate logins. It should also recognize 3 user roles: students, teachers, and administrators. With respect to the dictionary interface, only the administrators should be able to edit the dictionary on the site. Additionally both admins and teachers need to be able to create other users accounts, but while teachers can only create student accounts, admins can create and edit accounts of any time.

Users need to be able to edit their own account preferences. In the initial release the available customizations will relate primarily to how the dictionary is presented to them:

option description default

use 'v' Some books like to use a v to represent the consonant version of a u, others like to make Latin pointlessly harder to read. true
word click event Whenever any Latin text is displayed a user should be able to click on a word they do not know. When they make such a click, they may want to go to the full dictionary page, or simply to see an English translation of the word, and possible to see a Latin synonym of the word. These options should all be available. show English translation
search dictionary event Just as users can configure what they see when they click on a word, they should be able to customize the experience of searching the database. Our dictionary will story English translations, Latin synonyms, Latin definitions, and example sentences for each word. A user should be able to request that any of this information be shown or hidden show all

Administrative accounts need to be able to maintain the site's dictionary. To do this they need to create, edit and delete words from the lexicon. When an admin creates a word all regular, inflected forms should be added automatically, but the user needs to be able to change to any robot-generated information. Creating high-quality monolingual definitions will be very time consuming and so admins should be able to save incomplete entries; however, this also requires that they have easy access to the status of all dictionary entries to see what is and isn't finished

Anyone--whether logged in or not--should be able to search the dictionary. When a non-logged in user searches the dictionary, default preferences should be used. When a user types in a Latin word information for all possible lexemes which that word can be derived from will be returned. (E.g. 'amicis' is could be a form of either 'amicus' or 'amica' and therefore both definitions should be returned.) Additionally, the dictionary should be aware of alternative spellings of word (E.g. if 'lacryma' is searched the standared spelling 'lacrima' will be returned.)

When a user is reading a Latin definition of a word, they should be able to click on any of the words used in that definition and get the information specified in their user preferences for the clicked word

The option should also be available to search English translations of Latin words. If the dictionary is set in English-Latin mode, typing in an English will return all word for which that is a stored translation (E.g. typing in 'for' will return 'enim', 'pro', etc.).

Elementa Latina 0.1.1

Teachers can add their own words to private dictionaries. When a student account search for a word, any match in the site-wide dictionary or dictionaries associated with their teachers' accounts will be returned.

Elementa Latina 0.2

The second release will allow the site to be used for basic vocabulary training / assessment

Elementa Latina 0.3

The third release will introduce the site-wide library of Latin texts as well as teacher-specific libraries of readings for their classes

Elementa Latina 0.4

The fourth release will add the translation-grading mechanism.

Elementa Latina 0.5

The fifth release will implement the BlackBoard-lick features. Teachers will be able to create tests, quizzes and assignments to be automatically graded.

Elementa Latina 0.6

The sixth release will facilitate communication and file transfer between all types of user accounts.

Elementa Latina 1.0

The 1.0 release will indicate that the above features have been implemented and tested fully.
